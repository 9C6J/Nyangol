# Nyangol

마영전 냥골 던전 숙제 트래커입니다.  
`Nyangol.html` 단일 파일로 동작하며, 별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.

## 핵심 특징

- 로컬 전용 저장: 캐릭터/카운트/메모/가격/레이아웃/테마 정보가 브라우저 `localStorage`에 저장됩니다.
- 캐릭터 관리: 추가, 삭제, 상단 고정, 완료 체크, 순서 드래그 정렬.
- 카운트 관리: 주간/월간 카운트 증감, 개별 리셋, 전체 리셋.
- 정산표: 주간/월간 기준 보상 수량 및 골드 예상치 계산.
- 레이아웃 편집: 섹션 드래그 정렬 + 저장.
- 완료 연출 테마:
  - 전사형
  - 마법형
  - 궁수형
  - 도적형
  - 힐러형

## 완료 연출 테마 사용법

### 1) 캐릭터 추가 시 선택

`새 캐릭터 추가` 폼에서 이름 입력 후, 테마(`전사형/마법형/궁수형/도적형/힐러형`)를 고르고 추가하면 됩니다.

### 2) 이미 만든 캐릭터 테마 변경

캐릭터 카드의 `⚙️` 버튼을 누르면 테마 변경 프롬프트가 뜹니다.

- 입력 가능:
  - 번호: `1~5`
  - 키워드: `warrior`, `mage`, `archer`, `rogue`, `healer`

### 3) 완료 연출 발동 조건

- 주간 냥골 `10` 이상
- 월간 냥골 `30` 이상

둘 중 하나라도 달성하면 카드에 테마별 전용 연출이 표시됩니다.

## 실행 방법

1. `Nyangol.html` 파일을 브라우저로 엽니다.
2. 바로 사용하면 됩니다.

## 데이터 저장/주의사항

- 저장 위치: 현재 브라우저의 `localStorage`
- 같은 PC라도 브라우저가 다르면 데이터가 공유되지 않습니다.
- 브라우저 데이터 삭제 시 기록이 사라질 수 있습니다.

## 파일 구조

- `Nyangol.html`: 앱 전체(UI/CSS/JS 포함)
- `README.md`: 사용 안내 문서

## 커스터마이징 포인트

- 연출 테마 정의: `ANIMATION_THEMES`
- 완료 연출 DOM 생성: `createGoalCelebrationLayer(...)`
- 테마 변경 로직: `changeCharacterTheme(...)`
- 연출 스타일/키프레임: CSS의 `Goal Completion Celebration` 섹션
